생성자
 - 객체를 생성해주는 메서드(함수)
 - 생성자를 하나도 정의하지 않은 경우 컴파일러에서 자동으로 기본 생성자를 생성한다.
 - 클래스명과 동일한 명칭을 갖는다.
 - 생성자에 반환값을 정의 하지 못한다. -> 반환형을 정의하면 안된다.
 - 부가적으로 멤버변수의 초기화 해준다.
 - 기본 생성자 : 생성자가 아무것도 정의되어 있지 않은경우 컴파일러가 자동으로 추가해준다.
 - 생성자가 하나라도 정의되어 있으면, 기본생성자는 사라진다. -> 이러한 경우, 기본 생성자를 개발자가
   다시 정의 해줘야 한다.

오버로드
 - 생성자 오버로드 : 동일한 생성자를 매개변수만 다르게 하여 여러개 정의.
 - 메서드 오버로드 : 동일한 메서드를 매개변수와 반환값을 달리하여 여러개 정의.

접급제어자(추가 공부요)
 - 클래스, 멤버 변수, 메서드, 생성자
 - public > protected > default > private
 - 패키지 : 자바소스코드를 목적에 맞게 분류하여 관리하기 위한 것. (폴더)
 - public : 모든 패키지의 클래스 내부, 외부에서 접근이 가능하다.
 - protected : 동일 패키지 클래스 내부,외부에서 접근이 가능하고,
                 다만, 외부 패키지는 상속을 통해서 클래스 내부에서만 접근이 가능하다. -> 상속과 연관
 - default : 동일 패키지 클래스 내부,외부에서 접근이 가능하다.
 - private : 현재 클래스 내부에서만 접근이 가능하다.
